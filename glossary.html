用語集

Webブラウザ

Webページを閲覧するためのソフトウェアの事をWebブラウザと呼びます。
単にブラウザと呼ばれることもあります。

Webブラウザの例としては

  • Google Chrome
  • Edge
  • Internet Explorer
  • Safari
などがあります

Webページ

Webブラウザで閲覧している1ページの事をWebページと呼びます。
Webページは主にHTML・CSS・Java Scriptから構成されています。
また、Webページの集合体のことをWebサイトと呼びます。

WebページやWebサイトと似た言葉にホームページという言葉があります。
ほとんどWebページ・Webサイトと同じ意味でしようされますが、 厳密にはWebブラウザを起動した時に表示されるWebページのことをホームページと呼びます。

HTML

HyperText Markup Language の略です。
Webページの構造を定義するための言語です。
タグ（<>）と呼ばれる記号で文書に意味付け（マークアップ）をしていきます。
以下のような形式で書きます。

<開始タグ 属性名=“属性値”>コンテンツ</終了タグ>

HTMLのタグは基本的に開始タグと終了タグのセットになります。
タグによってどのような意味を持つかが決まります。
開始タグから終了タグまでの全体を要素と呼びます。
属性は、タグにつけるオプションで、タグによって設定できるものが決まります。
コンテンツには、文章や要素が入ります。

CSS

Cascading Style Sheets の略です。
Webページのレイアウトを整えるための言語です。
色の設定、サイズの設定、配置の調整などを、Webページの見た目に関する調整をします。

JavaScript

Webブラウザ上で動作するプログラミング言語です
Webページ上に、アニメーション的な動きを与えたり、ボタンをクリックした場合の処理、
キーボードでキーを入力した場合の処理などが定義することができます。

Webサーバー

作成したWebページをインターネット上に公開するためには、Webサーバーが必要になります。
Webサーバーは、WebブラウザからURL（リクエスト）を受け取り、
受け取ったURLに対応したWebページを返す（レンスポンスする）役割を持つソフトウェアです。
