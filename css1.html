CSSの基本を理解しよう

CSSの基本を理解するためのページです

CSSのではWebページのレイアウトを整えることができます

    ソースコード
    
        <!DOCTYPE html>
        <html lang="ja">
        <head>
            <meta charset="UTF-8">
            <title>CSSの基本</title>
            <style>
                /*
                セレクタ {
                    プロパティ: 値;
                }
                セレクタ：どの要素の
                プロパティ：何を
                値：どうするか
                */
                body {
                    text-align: center;　/* 全体を中央よせにします */
                }
        
                h1 {
                    background-color: pink;　/* h1の背景色をピンクにします */
                    color: blue;　/* 文字色を青にします */
                }
        
                /* idがexampleの要素が対象になります */
                #example {
                    color: red;
                }
        
                #btn {
                    font-size: 24px;　/* フォントサイズを24ピクセルにする */
                }
        
            </style>
        </head>
        <body>
            <h1>CSSの基本を理解しよう</h1>
            <p>CSSの基本を理解するためのページです</p>
            <p id="example">CSSのではWebページのレイアウトを整えることができます</p>
            <p><button type="button" id="btn">ボタン</button></p>
        </body>
        </html>
